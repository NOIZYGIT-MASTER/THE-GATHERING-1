## Metadata Standards

Every asset MUST include:

1. **WHO** - Creator credits (R.S. Plowman + any collaborators)
2. **WHY** - The story/intent behind creation
3. **WHAT** - Technical specifications
4. **WHEN** - Creation date/era
5. **WHERE** - Creation location/context
6. **HOW** - How creators can use it
