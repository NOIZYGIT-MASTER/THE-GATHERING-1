## How To Read The Map

Every region can be read through five layers:

1. Sonic anchors
2. State functions
3. Ritual archetype
4. DreamChamber biome
5. Interaction mode

This is not a flattening exercise.
It is a navigation layer that points toward deeper lineage-aware docs.

## State Routes

If a participant enters DreamChamber through feeling instead of geography, use these routes:

- `Activation`: West Africa, Latin America, Black Atlantic, festival and groove biomes
- `Focus`: Northern Asia, South Asia, East Asia, drone and overtone rooms
- `Reverence`: Himalaya, North Africa, orchestral/choral, bell and procession spaces
- `Lament`: Middle East, Persia, Celtic/Nordic, slow-string and reed corridors
- `Guidance`: Amazonia, Indigenous North America, voice-led ceremonial paths
- `Immersion`: Southeast Asia, Oceania, environmental and cyclical resonance worlds

## Practice Overlays

If a participant enters through method instead of region, use these overlays:

- `Drum Trance`: West Africa, North Africa, Northern Asia, Indigenous North America, Black Atlantic
- `Chant And Recitation`: South Asia, Himalaya, Europe, Middle East, North Africa
- `Overtone And Harmonic Voice`: Northern Asia, Himalaya, Arctic, experimental voice futures
- `Drone And Resonance`: South Asia, Oceania, Europe liturgical, Tibetan sphere, ambient/electronic futures
- `Procession And Movement`: North Africa, Middle East, Oceania, Europe, Latin America

These overlays are not permission to collapse cultures into one pool.
They are navigation routes for interaction design.

## Sonic Primitives

Across all lineages, DreamChamber keeps returning to the same building blocks:

- `pulse`
- `drone`
- `breath`
- `chant`
- `overtone`
- `rattle`
- `bell`
- `reed`
- `string shimmer`
- `environmental sound`

These are the lowest-level materials for biome design.

## Consciousness Levers

DreamChamber can modulate state through a small set of repeatable levers:

- `entrainment`: pulse, repetition, cycle
- `resonance`: overtone, drone, sustained vibration
- `somatic pressure`: bass, breath, bodily pacing
- `ritual framing`: threshold, transition, reflection, closure
- `group synchrony`: shared timing, collective response, circle logic

These levers should be explicit in prototype design.

## Build Implications

For product work, this map should drive:
