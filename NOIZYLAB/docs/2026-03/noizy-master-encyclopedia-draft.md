# NOIZY Master Encyclopedia Draft

## Summary

This is the first structured draft of the NOIZY Master Encyclopedia.

It is mined from the current canon and arranged so the system can be read in two ways:

- by project
- by date

It is not the final encyclopedia.

It is the starter ledger that makes the current NOIZY universe easier to browse, explain, and expand.

## Reading Rule

- dates below are current archive activity dates unless a more specific idea date is known
- one seed entry may point to several canonical files
- tags are starter navigation hints, not a final ontology
- this draft should be expanded by promoting more idea objects over time
