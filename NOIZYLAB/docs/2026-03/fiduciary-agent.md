# Fiduciary Agent

## Summary

The Fiduciary Agent is a human-supervised RightsOps system that monitors usage, prepares evidence, routes disclosures, and helps enforce the creator's rights.

## Version Strategy

### v0.1

- Monitor known platforms
- Compare releases against authorized assets
- Generate evidence packs
- Queue disputes and takedowns for review

### v0.2

- Quote licenses dynamically
- Route mentor-model access
- Score fraud and impersonation risk

## Inputs

- Provenance manifests
- Consent registry
- Asset fingerprints
- Platform policy matrix
- Beneficiary and payout rules

## Outputs

- Alert queue
- Evidence bundle
- Platform disclosure export
- Payout event routing

## Next Actions

- Define evidence-pack schema
- Define alert types and approval rules
