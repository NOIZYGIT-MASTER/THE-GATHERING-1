# Capture Inbox

Dump raw thoughts here first. Do not worry about polish.

## Capture Format

### Date

YYYY-MM-DD

### Project

Name the main system, product, or initiative.

### Title

Short name for the thought

### Idea

State the core thought in 1-3 lines.

### Detail

Freeform expansion, context, or examples.

### Impact

Why it matters.

### Related Ideas

List connected systems, files, or themes.

### Tags

Examples:

- noizy
- dreamchamber
- ai
- philosophy
- provenance
- product
- monetization
- partner
- legal
- research

### Promote To

Name the canonical file this should eventually live in.
