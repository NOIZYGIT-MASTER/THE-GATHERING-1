# NOIZY Master Encyclopedia

This archive is the canonical knowledge base for NOIZY vision, research, product specs, business logic, and partner-facing narratives.

Use it as a system, not a dump.

## Operating Rules

1. Capture raw thoughts in [07_INBOX/capture.md](./07_INBOX/capture.md).
2. Promote durable ideas into the correct volume.
3. Separate facts, ideas, speculation, and decisions.
4. Keep one concept per file whenever possible.
5. Update the relevant file instead of duplicating the same idea in multiple places.
6. Treat this index as the navigation spine for Claude or any other AI collaborator.

## Recommended Claude Workflow

1. Read this file.
2. Read only the specific files relevant to the task.
3. Summarize the current state before proposing changes.
4. Write updates back into the canonical file, not into chat-only drafts.

Suggested prompt:

```text
Read 00_INDEX.md, then read the files listed under the section I name.
Summarize the current state, identify contradictions, and propose one clean update.
Do not invent claims that are not supported by the archive.
```
