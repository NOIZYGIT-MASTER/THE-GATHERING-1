# Provenance Validator

## Summary

The Provenance Validator is the artist-facing and partner-facing interface that turns hidden lineage into visible trust.

## User Outcomes

- See whether a work is safe to release
- See what AI touched the work
- See whether source assets were consented
- Export audit or evidence bundles

## UI Layers

### Public Layer

- Human Contribution Trace
- Consent-Verified Source
- AI-assisted role labels
- Commercial-Use Readiness

### Audit Layer

- Asset IDs
- Source lineage
- Training eligibility
- Model and version IDs
- Evidence bundle exports

## Risks

- Overpromising verification where data is only self-reported
- Using labels that cannot be defended
