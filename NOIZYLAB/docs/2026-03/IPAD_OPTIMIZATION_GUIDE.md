# iPAD PRO 12.9 - PERFORMANCE OPTIMIZATION
## GORUNFREE - Maximum Speed & Response

---

## IMMEDIATE IMPACT SETTINGS

### 1. REDUCE MOTION & ANIMATIONS
**Settings → Accessibility → Motion**
- ✓ Reduce Motion: **ON**
- ✓ Prefer Cross-Fade Transitions: **ON**
- ✓ Auto-Play Message Effects: **OFF**
- ✓ Auto-Play Video Previews: **OFF**

*Cuts animation lag by 50%+*

---

### 2. DISABLE TRANSPARENCY
**Settings → Accessibility → Display & Text Size**
- ✓ Reduce Transparency: **ON**
- ✓ Reduce White Point: **ON** (optional, saves battery)

*Removes GPU-heavy blur effects*

---

### 3. BACKGROUND APP REFRESH
**Settings → General → Background App Refresh**
- Set to: **OFF** or **Wi-Fi only**
- Or disable per-app (keep only essential)

*Frees RAM and CPU*

---

### 4. LOCATION SERVICES CLEANUP
**Settings → Privacy & Security → Location Services**
- Set apps to: **While Using** (not Always)
- Disable for apps that don't need it
- System Services → Disable:
  - Analytics & Improvements
  - iPhone Analytics
  - Routing & Traffic
  - Improve Maps

---

### 5. SIRI OPTIMIZATION
**Settings → Siri & Search**
- ✓ Listen for "Hey Siri": **ON** (for voice control)
- ✓ Press Side Button for Siri: **ON**
- Siri Suggestions: **OFF** for most apps
- Show in Look Up: **OFF**
- Show in Spotlight: **OFF**

*Keeps Siri fast, removes indexing overhead*

---

### 6. SPOTLIGHT SEARCH CLEANUP
**Settings → Siri & Search**
Scroll down - disable search for apps you don't search:
- Only enable for apps you actually search
- Disable: Games, rarely used apps

*Reduces constant indexing*

---

### 7. MAIL OPTIMIZATION
**Settings → Mail**
- Fetch New Data: **Manually** or **Hourly**
- Push: **OFF** (uses constant connection)

**Settings → Mail → Accounts → Fetch New Data**
- Set to Manual or longer intervals

---

### 8. CLEAR SAFARI
**Settings → Safari**
- Clear History and Website Data
- Close All Tabs (or limit to 100)
- Disable: Preload Top Hit
- Disable: Search Engine Suggestions

---

### 9. STORAGE CLEANUP
**Settings → General → iPad Storage**

**Quick wins:**
- Delete unused apps (or offload)
- Clear Messages attachments
- Delete old photos/videos
- Review "Recommendations" at top

**Offload Unused Apps:**
**Settings → App Store**
- ✓ Offload Unused Apps: **ON**

*Removes app but keeps data*

---

### 10. KEYBOARD OPTIMIZATION
**Settings → General → Keyboard**
- Disable: Predictive (if using voice)
- Disable: Slide to Type (if not using)
- Disable: Enable Dictation (re-enable = fresh cache)

*Then re-enable Dictation*

---

## DEEP CLEAN

### RESTART iPAD
Hold Power + Volume → Slide to Power Off → Wait 30 sec → Power On

*Clears RAM, resets caches*

---

### RESET ALL SETTINGS (Nuclear Option)
**Settings → General → Transfer or Reset iPad → Reset → Reset All Settings**

- Keeps your data/apps
- Resets all preferences to default
- Often fixes mysterious slowdowns

---

### NETWORK RESET
**Settings → General → Transfer or Reset iPad → Reset → Reset Network Settings**

- Clears all WiFi passwords
- Fixes connectivity issues
- Re-enter WiFi after

---

## ACCESSIBILITY PERFORMANCE BOOSTS

### For Voice Control Users:
**Settings → Accessibility → Voice Control**
- ✓ Voice Control: **ON**
- Commands: Only enable what you use
- Vocabulary: Clear unused custom words

### Switch Control (if not using):
**Settings → Accessibility → Switch Control**
- ✓ Switch Control: **OFF**

### AssistiveTouch Optimization:
**Settings → Accessibility → Touch → AssistiveTouch**
- If enabled, minimize custom actions
- Reduce menu items for speed

## APP-SPECIFIC OPTIMIZATION

### TERMIUS (SSH)
- Settings → Clear cache
- Reduce saved sessions if over 50
- Disable sync if not needed

### CLAUDE APP
- Clear conversation history periodically
- Don't keep 100+ open chats

### SAFARI
- Use Reader Mode for heavy pages
- Enable Content Blockers (ad blocker)
- Limit open tabs to <20

### FILES APP
- Remove unused cloud connections
- Clear Recently Deleted

## iPadOS 17/18 SPECIFIC

### STAGE MANAGER (if enabled)
**Settings → Home Screen & Multitasking**
- If laggy: **Disable Stage Manager**
- Or limit to 3 windows max

### WIDGETS
- Remove widgets you don't check daily
- Each widget = background refresh

### LOCK SCREEN
- Remove widgets from lock screen
- Simpler = faster wake
