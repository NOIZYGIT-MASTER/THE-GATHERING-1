# Monetization Models

## Summary

NOIZY monetization should compound creator value instead of reducing artists to one-off sales.

## Revenue Modes

- Direct releases
- Opt-in training datasets
- Inference usage
- Voice actor twin licensing
- Directed live voice sessions
- Mentor models
- Taste and curation channels
- Artifact editions
- Licensing for film, games, and interactive media

## Principles

- Every revenue event must map to a consent scope.
- Every payout must be auditable.
- Beneficiary and estate logic should exist from the start.
- Dashboards should make recurring value legible to the artist.

## Voice Twin Economy

Voice actors should be able to treat a home studio as a governed long-term income engine.

That means:

- visible lifetime value
- use-case aware pricing
- actor veto power over categories
- recurring twin revenue without surrendering source control
