## The NOIZYVOX Framework

NOIZYVOX is not a voice cloning product. It is a **creative economy protocol**.

### Core Structure

```
Voice Actor → Records Character Profiles → Signs NOIZYVOX Consent Agreement
                                                    ↓
                            Voice DNA captured, fingerprinted, SHA-locked
                                                    ↓
                    75% Perpetual Royalty | 25% Platform (NOIZY/Anthropic split)
                                                    ↓
                         Every licensed use traces back to the source voice
                                                    ↓
                    Voice assets can be gifted, inherited, or willed — generational wealth
```

### What "Perpetual" Means

This is not a streaming rate. This is not a one-time license fee.

Every time a voice derived from an artist's profile is used — in a product, in a game, in a film sync, in a conversational AI — a micro-split fires. The artist's estate owns it. Their children own it. The original consent is locked in the Vault. It cannot be reversed.

This is the LifeLUV model applied to voice: **a creative asset that compounds over time rather than depreciating**.
