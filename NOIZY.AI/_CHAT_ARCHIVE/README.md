# NOIZY.AI Chat Archive

Purpose: intake, sort, and preserve exported chat material for THE-GATHERING.

## Boundary

Copilot cannot directly access or control a user's chat sidebar history. Chat material must be exported, copied, or saved into `_UNSORTED_INBOX/` before it can be organized.

## Folder rules

- `_UNSORTED_INBOX/` — drop raw exported/copied chats here.
- `_EMPTY_OR_DUPLICATE/` — placeholder for chats confirmed empty or duplicated.
- `_INDEX/` — generated indexes and summaries.
- `YYYY/` — organized dated archives.

## Naming rule

Final chat files should use:

```text
YYYY-MM-DD_PROJECT_TOPIC.md
```

## Safety rule

Do not store passwords, API tokens, recovery codes, private contact IDs, or secret keys in this archive.
