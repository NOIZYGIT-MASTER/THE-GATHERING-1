Using RX as an audio editor with Logic Pro X

RX is a powerful audio editor that Apple Logic Pro X users can use to get better sounding audio. To use RX with Logic, you must first set it up as an external audio editor.

Setting up as an external audio editor

  1. Open Preferences>Advanced
  2. Under Additional options, enable Audio
  3. Open Preferences>Audio
  4. Under Audio File Editor, click on the External Sample Editor to select RX 5 from your applications folder

Workflow

  1. Select the clip you wish to edit in your timeline
  2. Click Edit>Open in iZotope RX 5 Audio Editor, [Shift+W]
  3. File will open in RX 5. Once you’ve completed your edits, click File>Overwrite Original File
  4. Close tab, navigate back to Logic Pro X and wait for waveform to update
