Overview

The Gate is a welcomed addition to Neutron’s next-generation channel strip workflow. It features all of the controls of a traditional Gate in addition to brand new Hysteresis and Hold DSP - all built from the ground up. With multiband functionality, you can apply frequency dependant Gate control, or creatively duck any part of the spectrum using the expansive sidechain controls featured in every band.

Controls

Threshold

Determines the level that will trigger the gate to open or close. Signals that fall below the threshold will trigger the gate to close, signals exceeding the threshold will trigger the gate to open.

Hysteresis

Functions as a second threshold for gating. The Hysteresis threshold can be thought of as the “close” threshold, in that signals that fall below this value will close the gate. Signals that exceed the Hysteresis threshold will not open the gate until they exceed the Threshold value.

The Hysteresis control has a range of 0dB to 30dB. Setting Hysteresis to 30 will place the second threshold 30 dB lower than the current Threshold setting. Setting Hysteresis to 0 will effectively disable Hysteresis, as it will equal the Threshold value.

Set the threshold so that it gates all unwanted signals, then turn up the Hysteresis value to reduce unwanted gating.

Hold

Sets the amount of time the gate will stay fully open after a signal falls below the threshold. The hold length will vary when the release period begins.

Ratio

Determines the amount of gain reduction applied to signals that fall below the Threshold.

Attack

Determines the length of time it takes for the gate to transition from closed (silenced) to open (no gating) when a signal exceeds the threshold.

Release

Determines the length of time it takes for the gate to transition from open (no gating) to closed (silenced) when a signal falls below the threshold.

Add/Remove band

Adds or remove the associated band.

Bypass

Bypasses processing for the associated band.

Solo

Solos the output of the associated band.

Sidechain Enable/Disable

Enables/disables sidechain input for the current band.

Sidechain Source

Allows you to select the sidechain source for the associated band. You can choose between internal or external bands.

Component Plug-in

The Gate is available as a component plug-in with Neutron 2 Advanced.
