Relay

English 日本語

Overview

Relay can send metering data to and receive information from select Inter Plug-in Communication (IPC) compatible iZotope plug-ins, including: Insight 2, Tonal Balance Control, Neutron 3, Neutron 3 EQ plug-in, and Neutron 3 Visual Mixer.

Want to Know More About Relay?

For more information about Relay, click on the “?” (Help) button in the Relay plug-in to access Relay’s Help Documentation.

Relay and Neutron 3 Masking Meter

Relay can be selected as a source in the Masking Meter included in Neutron 3 (EQ module) and the Neutron 3 EQ component plug-in.

Relay and Neutron 3 Visual Mixer

Relay controls can be remotely modified in the Neutron 3 Visual Mixer. Moving a Relay node up or down in the Visual Mixer will adjust the gain slider in the remote instance. Moving a Relay node left or right in the Visual Mixer will adjust the pan control in the remote instance (stereo Relay instances only). Dragging Relay node handles in the Visual mixer will adjust the width parameter in the remote instance (stereo Relay instances only).

Relay and Neutron 3 Mix Assistant

With the new Mix Assistant feature, we recommend a workflow using Relay in your session. For more information on this workflow, see the Assistants chapter.

Relay and Tonal Balance Control

Relay can be selected as a source in the Tonal Balance Control plug-in.
