Vintage Compressor

Overview

The Vintage Compressor is an emulation of a feedback compressor, and includes a detection filter in the feedback loop. In a feedback compressor, level detection occurs at the output of the compressor, as opposed to the input.

Channel Processing Modes

The Vintage Compressor module can operate in either Stereo or Mid/Side mode. See the Channel Processing section for more details.

Reset

Returns all controls in the Vintage Compressor module to their factory default values.

UNDO HISTORY TO REVERT SETTINGS

If you wish to return to settings you were using before clicking the Reset button, you can use the Undo History window to revert the settings.

Controls

The following controls are available in the Vintage Compressor Control Panel:

  • Mode
  • Threshold
  • Ratio
  • Attack
  • Release
  • Makeup Gain
  • Auto Gain

Mode

Adjusts the overall character of the Vintage Compressor. For more detailed information on the modes, see the table below.

MODE EFFECT
Sharp Provides crisp dynamics and a greater emphasis on transients while maintaining the body of the signal.
Balanced Gives a signal-dependent balance between dynamics preservation and overall enhancement to the body of the signal.
Smooth Smoothes out transient and dynamic material while enhancing and bringing out the rest of the signal, resulting in a thicker and fuller sound.

Threshold

Adjust the threshold to set the point where the dynamics processing takes place. Since some modes have a soft knee, mild compression may occur below this point.

Threshold Meter

Displays the input level to the module alongside the gain reduction applied by the module’s processing.

  • The two meters on the far left and right of the threshold meter display the input audio level.
  • The two meters in between the input meters display the gain reduction applied.

Ratio

Sets the amount of attenuation applied to the signal once it has passed the threshold. Higher ratios will result in more compression.

Attack & Release

Adjust the attack and release controls to set how quickly the Vintage Compressor reacts to audio that crosses the threshold.

  • ATTACK: Determines how quickly the Vintage Compressor reacts when the threshold is reached.
  • RELEASE: Determines the amount of time before the Vintage Compressor returns the level. This control sets the release time for transients. For sustained compression, the release time will be significantly longer.

Qualities of Attack and Release

The Attack and Release express unique qualities in a the vintage compressor, such as:

  • Release time on transients is much faster than it is on periods of sustained compression, which helps to decrease both pumping and distortion.
  • Attack and release times can vary depending on how much compression is occurring.

Gain

Adjusts the output gain of the Vintage Compressor module to makeup for decreases in volume.

Auto Gain

When turned on, Auto Gain calculates the RMS levels of both the input and output signals of the Vintage Compressor in order to apply the appropriate gain to the output signal.

Auto Gain as A/B Tool

Auto Gain acts as a smart “make-up gain” control that adapts over time and adjusts your levels to a comparable level of your unprocessed audio. This can be a useful tool for A/B’ing various settings without having gain changes affect your perception.
