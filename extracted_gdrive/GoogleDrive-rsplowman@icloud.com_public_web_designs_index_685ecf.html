Spectral Shaper

ADV

Overview

The Spectral Shaper can be used to apply high-resolution attenuation of problematic frequencies across the full spectrum with configurable time constants, timbre adjustment, and a variable full spectrum action region.

Module Interface

Ozone 9’s Spectral Shaper module controls and features are outlined in the table below.

View Selector

Spectral Shaper module includes the following views:

  • Spectrum Analyzer
  • Gain Trace

Channel Processing Modes

The Spectral Shaper module can operate in either Stereo or Mid/Side mode. See the Channel Processing section for more details.

Reset

Returns all controls in the Spectral Shaper module to their factory default values.

UNDO HISTORY TO REVERT SETTINGS

If you wish to return to settings you were using before clicking the Reset button, you can use the Undo History window to revert the settings.

Controls

There are two sets of controls you can adjust within the Spectral Shaper module:

  • Range Selectors
  • Control Panel

Range Selectors

Adjust the area of applied processing, or Action Region, in the Spectrum view by clicking on a Range Selector and dragging the Range Selector left or right across the display.

ACTION REGION WIDTH EFFECT
Narrow Action Region You can surgically apply spectral shaping to meet your target curve on a specific range.
Wide Action Region You can address coloration similar to an EQ by applying processing on the whole spectrum to adjust overall balance.

Adjusting Range Selectors

You can adjust the Range Selectors using the methods described in the table below.

TYPE OF ADJUSTMENT DESCRIPTION
One Range Selector Click on the Range Selector handle and drag to the desired frequency.
Both Range Selectors Click between the Range Selectors and drag both cutoff handles together maintaining the width between the Range Selectors.
Text Readout Double-click on the text readouts above the Range Selector handles and enter a new value in the text edit box.

Action Region Solo

Solos the input signal before applied processing within the designated Action Region.

Control Panel

Spectral Shaper includes the following controls in the control panel:

  • Modes
  • Threshold
  • Character

Modes

Sets the intensity of reduction applied by the Spectral Shaper.

MODES DESCRIPTION
LIGHT Light amount of processing applied.
MEDIUM Medium amount of processing applied.
HEAVY Heavier amount of processing applied.

Threshold

Adjust to set the level at which spectral gain reduction is applied to the action region. The threshold control includes the following sub-controls and meter:

  • Threshold Input Meter
  • Listen
Threshold Input Meter

Displays the highest level of all bands included in the Action Region range.

  • The two meters on the far left and right of the threshold meter display the input audio level.
  • The two meters that appear in between the input meters display the gain reduction applied.

Spectral Shaper Threshold Input Meter

Spectral Shaper’s Threshold Input Meter displays different information than the threshold input meter in the Maximizer module or Vintage Limiter module. In the Maximizer and Vintage Limiter modules, the Threshold input level is based on the full frequency spectrum. The Spectral Shaper Threshold input level is based on the highest level of the frequency bands included within the action region.

Listen

Enable to solo the processed (reduced) audio related to the Threshold being triggered.

Tone

Adjust to set the spectral tilt of the Spectral Shaper processing.

TONE EFFECT
Positive Values Tilt towards a brighter overall spectral character.
Negative Values Tilt towards a darker overall spectral character.

Attack

Adjust to set the amount of time it takes for the Spectral Shaper to apply gain reduction to signals that exceed the Threshold.

Release

Adjust to set the amount of time it takes for the Spectral Shaper to stop applying gain reduction to signals that fall below the Threshold.
