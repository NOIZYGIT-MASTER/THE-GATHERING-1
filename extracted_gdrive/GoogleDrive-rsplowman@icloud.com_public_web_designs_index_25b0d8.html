Codec Preview

ADV

Overview

When converting to a lossy audio format (such as MP3 or AAC), psychoacoustic algorithms are used to truncate the less audible components of your audio. This means that parts of your audio are actually being removed, which has a subtle (or obvious, at lower bit rates) affect on the sound.

Codec Preview allows you to hear what your master will sound like after lossy audio compression has taken place, and gives you the opportunity to compensate for this data loss without leaving your DAW. For example, when converting to a low bit rate MP3, you may want to brighten your audio slightly, to counteract the loss of high frequency data.

Accessing Codec Preview

The Codec Preview panel can be accessed by clicking the Codec button in the Auditioning section of the Ozone application and the main Ozone 8 plug-in.

Codec Preview Availability

Codec Preview is an Advanced only feature and is not available in the component plug-ins. You can access Codec Preview in the main Ozone plug-in and Ozone application.

Signal Flow

Codec Preview is at the very end of the signal chain within Ozone. When you are using Ozone as a plug-in, it is important that Ozone is placed after all other inserts if you are using Codec Preview. This will ensure that the signal going into Codec Preview includes all of the processing being applied to the mix.

Codec Preview is intended to be used for previewing only and will not apply actual MP3 or AAC compression when exporting from the Ozone application or a DAW. You can use the Codec Preview feature to help inform the export settings of your host application, but be sure to disable the feature prior to exporting When you enable Codec Preview, codec preview processing is reflected in the:

  • I/O Panel output meters: monitor clipping that might occur as a result of lossy compression.
  • Spectrum meters: useful for monitoring the effect of lossy compression with regard to frequency, specifically, in the high end when auditioning lower bit rates.

Controls

The Codec Preview panel includes the following controls:

  • Format
  • Bit Rate
  • Solo Artifacts

Format

Select the codec format to preview: MP3 or AAC.

OZONE CODEC FORMAT INFORMATION

  • AAC: Ozone uses the Fraunhofer AAC codec.
  • MP3: Ozone uses the LAME codec www.mp3dev.org.

Sample Rates & Performance

AAC and MP3 files do NOT support sample rates greater than 48kHz.

  • If your session sample rate is set to a value other than 44.1kHz or 48kHz, Ozone will perform high-quality real-time resampling when you enable Codec Preview.
  • Real-time resampling requires high CPU usage. This may incur significant latency especially within high sample rate sessions that utilize small buffer sizes. Most hosts should be able to compensate for this latency, but you may notice a slight delay in playback when Codec Preview is enabled.
  • We recommend adjusting the buffer size to a larger value to avoid any discontinuities or performance issues when using Codec Preview within high sample rate sessions.

Bit Rate (Constant)

Sets the constant bit rate value used by Codec Preview.

BIT RATE OPTIONS

112 kbps
128 kbps

MONO TRACKS AND BIT RATES

Ozone 9 offers a range of bit rates. If you are working on a mono track, Codec Preview will only support preview of 96 kbps through 256 kbps.

Solo Artifacts

Enable to hear the audio data being lost (artifacts) in the codec conversion.

CODEC ARTIFACT SIGNAL DERIVATION

The codec artifact signal is the codec preview signal subtracted from the unencoded signal.
