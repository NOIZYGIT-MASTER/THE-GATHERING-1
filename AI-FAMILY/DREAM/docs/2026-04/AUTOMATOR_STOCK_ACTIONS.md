### 1. AI Processing Pipeline
```
Ask for Text → Run Shell Script (curl to API) → Display Notification
```

```

```
Get Folder Contents → Filter Finder Items → Run Shell Script → Display Notification

## 🎮 QUICK START RECIPES

### Basic Shell Command
1. Drag "Run Shell Script" action
2. Set shell to `/bin/zsh`
3. Pass input: "as arguments"
4. Add your command

### API Call Template
1. "Get Specified Text" (API endpoint)
2. "Run Shell Script" (curl command)
3. "Display Notification" (result)

### File Processor
1. "Ask for Finder Items"
2. "Filter Finder Items" (by type)
3. "Run Shell Script" (process)
4. "Move Finder Items" (to processed folder)
