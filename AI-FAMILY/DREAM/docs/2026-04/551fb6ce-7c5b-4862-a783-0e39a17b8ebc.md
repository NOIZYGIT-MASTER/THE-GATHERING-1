## The Broadcast Pattern

Instead of per-app mic integration fights:

```
Mac Studio Mic
      ↓
Local Audio Server (Python, port 8765)
      ↓  WebSocket broadcast
  ┌───┴───┬────────┬──────────┐
  │       │        │          │
VS Code  GABRIEL  Talon     MC96
Ext      Daemon   Script    Portal
```

One mic. One capture process. Every consumer subscribes. No conflicts, no licensing, no Rōba needed.

## It's ~30 lines of Python

```python
# audio_broadcast.py
import asyncio, websockets, sounddevice as sd
import numpy as np, json

CLIENTS = set()
SAMPLE_RATE = 16000

async def register(ws):
    CLIENTS.add(ws)
    try: await ws.wait_closed()
    finally: CLIENTS.discard(ws)

def mic_callback(indata, frames, time, status):
    pcm = indata[:, 0].tobytes()
    asyncio.get_event_loop().call_soon_threadsafe(
        lambda: [asyncio.ensure_future(c.send(pcm)) for c in CLIENTS]
    )

async def main():
    with sd.InputStream(samplerate=SAMPLE_RATE, channels=1, callback=mic_callback):
        async with websockets.serve(register, "localhost", 8765):
            await asyncio.Future()  # run forever

asyncio.run(main())
```
