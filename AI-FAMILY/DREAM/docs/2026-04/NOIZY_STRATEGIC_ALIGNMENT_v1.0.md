## 1. The 9-Layer Competitive Moat

NOIZY's defensibility is not a single feature — it is a layered architecture where each layer reinforces the others. No competitor can replicate the full stack without the doctrine.

| L2 | **Voice Estate (HVS token)** | AI labs that claim voice samples as training data |

| L5 | **Never Clauses (9 immovable)** | Race-to-bottom platform terms |
| L6 | **Guild of Artists collective** | Individual creator negotiating weakness |

| L8 | **GABRIEL agentic royalty routing** | Human intermediary fee layers |
| L9 | **Open protocol (NCP = TCP/IP)** | Proprietary lock-in by competitors |

### Why This Moat Compounds

Each layer is individually strong. Together they are self-reinforcing:
- L1 (consent) feeds L4 (audit) — every consent event is logged
- L3 (C2PA) feeds L2 (HVS) — every audio file points back to its voice estate
- L8 (GABRIEL) feeds L7 (75/25) — automatic routing enforces the split
- L9 (open protocol) feeds L6 (Guild) — any creator can join, strengthening collective

A competitor building only L1 (consent forms) has no protection against L3 (provenance) bypass. NOIZY is the only system where all 9 layers are unified.
