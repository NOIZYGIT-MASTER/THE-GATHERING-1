# DreamChamber Imperial Charter — Daily Operation

The empire runs in four modes.
The mode is set by energy, not by calendar.

---

## Red Mode — Low Energy

Read-only. No building.

Permitted actions:
- iPhone probe
- Open receipts
- Review manifests
- Logic Remote transport only
- Claude explains, does not refactor

Accessibility stack: Voice Control + Action Button only

---

## Yellow Mode — Medium Energy

Operate and understand.

Permitted actions:
- Raycast health checks
- One function review
- One bug inspection
- One archive pass
- One transport or mix tweak

Accessibility stack: Voice Control + Keyboard + Logic Remote

---

## Green Mode — High Energy

Build.

Permitted actions:
- Claude Code active in VS Code (auto mode ON)
- AUv3 / Worker / Receipt / UI work
- One focused mission only — no scope drift

Accessibility stack: Full stack — Voice + Keyboard + Touch + Logic Remote

---

## Gold Mode — DreamChamber Full Boot

Deploy and orchestrate.

Permitted actions:
- Full build session
- HEAVEN checks and deployment operations
- Logic + IDE + notes + mission board active
- Real output with receipts emitted

Accessibility stack: All devices active — GOD + iPad + iPhone

---

RSP_001 speaks

## Keyboard command hierarchy

```
Comfort → Voice → Keyboard → Touch → Mouse

Cmd+Space  → Raycast command palette
Cmd+P      → navigate
Cmd+Shift+P → act
Cmd+`      → terminal
Shift+Tab  → cycle Claude Code permission mode (to Auto)
```
