# First-time setup
./audio-control.sh --setup

# Daily operations

./audio-control.sh --mute-all  # kill all mics
./audio-control.sh --logitech  # go live: Logitech

./audio-control.sh --status    # full status

Setup Checklist
One-time configuration on M2 Ultra
  1. Install Homebrew (if not already installed)
  2. Run brew install switchaudio-osx
  3. Run brew install --cask hammerspoon
  4. Copy hammerspoon-init.lua → ~/.hammerspoon/init.lua
  5. Copy audio-control.sh to your preferred location
  6. Run chmod +x audio-control.sh
  7. Run ./audio-control.sh --setup (installs login agent)
  8. Run ./create-aggregate-device.sh to combine both mics
  9. Open Hammerspoon → Reload Config
  10. Test: Cmd+Shift+S should show status overlay
