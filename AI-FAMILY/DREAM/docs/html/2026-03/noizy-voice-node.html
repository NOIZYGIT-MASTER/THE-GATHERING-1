Architecture

Service Overview

The voice node is the hardware gateway into the NOIZYVOX ecosystem. It runs as a persistent Node.js daemon on the MacBook Pro connected to the UAD interface. Everything that enters the pipeline — every syllable Rob speaks into the U87 — is governed by consent keys and Never Clauses before a single byte touches the network.

Data Flow

Pipeline Stages

Every utterance traverses all seven stages in sequence. Consent failure at Stage 1 is permanent — the utterance is dropped and logged, never held in memory. No utterance exits the node without a valid watermark hash and complete provenance record.

01
Core Audio Capture
ffmpeg/AVFoundation samples UAD interface at 48kHz/24-bit. Emits 100ms PCM chunks to VAD queue.
↓
02
Pre-buffer Rolling Window
300ms circular pre-buffer maintains audio before speech onset. Ensures attack transients are never clipped.
↓
03
Voice Activity Detection
RMS energy per chunk vs −40 dBFS threshold. State machine: silence → speech → speech_ending. 800ms silence padding. Min utterance: 500ms.
↓
04
Consent Key Validation
NOIZY-{ACTORID}-CONSENT-{UUID}-{AVA}-{DOMAIN}-{JURISDICTION}-{DATE} format check. If invalid → DROP + log. Never Clause checks run in parallel.
↓
05
WAV Write
PCM chunks assembled with correct 44-byte RIFF header (44.1k/48k/24-bit). Saved to /tmp/noizyvox/utterances/ with utterance ID filename.
↓
06
Watermark Hash
Metadata watermark: actorId + utteranceId + sessionId + consent key prefix + timestamp → deterministic WM_{hex8} hash. Future: WavMark audio steganography.
↓
07
Dispatch + Provenance Ledger
POST to NOIZYVOX ingest API (or local stub). Full ProvenanceRecord appended to provenance-ledger.jsonl — immutable, append-only, never overwritten.
Never Clause Enforcement: NC_01 through NC_10 are checked at Stage 04 before any data leaves the capture buffer. The consent key must be valid for the specific AVA, domain, and jurisdiction encoded in the key. A denial is logged but the audio is never transmitted.
Installation

Setup Guide

Prerequisites

Install Node.js 20+, ffmpeg (via Homebrew), and TypeScript toolchain on the MacBook Pro.

$brew install ffmpeg node
$node --version # must be v20+
$ffmpeg -version # confirms AVFoundation support

Connect UAD interface via USB or Thunderbolt. Open UAD Console and confirm the Neumann U87 preamp is active on Input 1.

Install from NOIZY Monorepo

Clone the noizy-platform repo and navigate to the voice-node service. Install dependencies.

$cd noizy-platform/services/voice-node
$npm install
$npm run build

Discover Your UAD Device Name

Run the device list command to find the exact Core Audio device string for your UAD hardware. The name varies by model and macOS version.

$npm run list-devices

Copy the exact device name from the output (e.g. Apollo Twin USB) and update defaultConfig.audio.deviceName in config.ts.

Configure Environment Variables

Create .env in the service root. Minimum required: consent key and actor ID.

NOIZY_CONSENT_KEY=NOIZY-RSP001-CONSENT-7a4f2b8e9d3c1f6a-AVA001-NARR-CA-ON-2026-03-13
NOIZYVOX_ENDPOINT=http://localhost:3001
DEBUG=false

Start in development mode. Speak into the U87. The terminal will show VAD state in real time. Ctrl+C for graceful shutdown.

capture.ts

CaptureService wraps ffmpeg's AVFoundation input to stream raw PCM from the UAD interface. Emits precisely-sized AudioChunk events at bufferSizeMs intervals with timestamp, sample rate, and bit depth metadata attached. Handles device discovery, remainder buffering for exact chunk boundaries, and graceful SIGTERM shutdown.

src/capture.ts TypeScript 
// NOIZYVOX Voice Node — Core Audio Capture Service
// ffmpeg + AVFoundation → precise PCM chunk stream

import { spawn, ChildProcess } from 'child_process';
import { EventEmitter } from 'events';
import { AudioConfig } from './config';

export interface AudioChunk {
  data:        Buffer;
  timestamp:   number;    // ms since session start
  durationMs:  number;
  sampleRate:  number;
  channels:    number;
  bitDepth:    number;
}

export class CaptureService extends EventEmitter {
  private config:    AudioConfig;
  private process:   ChildProcess | null = null;
  private isRunning: boolean = false;
  private startTime: number  = 0;

  constructor(config: AudioConfig) {
    super();
    this.config = config;
  }

  // ── List available Core Audio devices via ffmpeg ─────────────────
  async listDevices(): Promise<string[]> {
    return new Promise((resolve, reject) => {
      const proc = spawn('ffmpeg', [
        '-f', 'avfoundation', '-list_devices', 'true', '-i', '""'
      ], { stdio: ['ignore', 'ignore', 'pipe'] });

      let stderr = '';
      proc.stderr?.on('data', (d) => stderr += d.toString());
      proc.on('close', () => {
        const lines = stderr.split('\n');
        const audioDevices: string[] = [];
        let inAudio = false;
        for (const line of lines) {
          if (line.includes('AVFoundation audio devices')) inAudio = true;
          if (inAudio && line.match(/\[\d+\]/)) {
            const match = line.match(/\[(\d+)\] (.+)/);
            if (match) audioDevices.push(`[${match[1]}] ${match[2].trim()}`);
          }
        }
        resolve(audioDevices);
      });
      proc.on('error', reject);
    });
  }

  // ── Find device index for configured device name ──────────────────
  async findDeviceIndex(): Promise<string> {
    const devices = await this.listDevices();
    const target  = this.config.deviceName.toLowerCase();
    for (const device of devices) {
      if (device.toLowerCase().includes(target)) {
        const match = device.match(/\[(\d+)\]/);
        if (match) return match[1];
      }
    }
    console.warn(`[CAPTURE] Device "${this.config.deviceName}" not found. Using default input.`);
    return '0';  // :0 = default input
  }

  // ── Start capture stream ──────────────────────────────────────────
  async start(): Promise<void> {
    if (this.isRunning) throw new Error('Capture already running');

    const deviceIndex  = await this.findDeviceIndex();
    const chunkSamples = Math.floor(
      this.config.sampleRate * this.config.bufferSizeMs / 1000
    );

    // ffmpeg: AVFoundation audio → raw PCM → stdout
    const args = [
      '-hide_banner', '-loglevel', 'error',
      '-f', 'avfoundation',
      '-sample_rate', String(this.config.sampleRate),
      '-i', `none:${deviceIndex}`,            // none = no video
      '-f', `s${this.config.bitDepth}le`,     // signed PCM LE
      '-ac', String(this.config.channels),
      '-ar', String(this.config.sampleRate),
      '-blocksize', String(
        chunkSamples * this.config.channels * (this.config.bitDepth / 8)
      ),
      'pipe:1',
    ];

    this.process  = spawn('ffmpeg', args, { stdio: ['ignore', 'pipe', 'pipe'] });
    this.isRunning = true;
    this.startTime = Date.now();

    this.process.stderr?.on('data', (data) => {
      const msg = data.toString();
      if (msg.toLowerCase().includes('error')) {
        this.emit('error', new Error(`ffmpeg: ${msg.trim()}`));
      }
    });

    // ── Exact-size chunking with remainder accumulator ─────────────
    const bytesPerSample = this.config.bitDepth / 8;
    const bytesPerChunk  = chunkSamples * this.config.channels * bytesPerSample;
    let remainder        = Buffer.alloc(0);

    this.process.stdout?.on('data', (data: Buffer) => {
      const combined = Buffer.concat([remainder, data]);
      let offset = 0;
      while (offset + bytesPerChunk <= combined.length) {
        const chunk = combined.slice(offset, offset + bytesPerChunk);
        this.emit('chunk', {
          data:       chunk,
          timestamp:  Date.now() - this.startTime,
          durationMs: this.config.bufferSizeMs,
          sampleRate: this.config.sampleRate,
          channels:   this.config.channels,
          bitDepth:   this.config.bitDepth,
        } as AudioChunk);
        offset += bytesPerChunk;
      }
      remainder = combined.slice(offset);
    });

    this.process.on('close', (code) => {
      this.isRunning = false;
      if (code !== 0 && code !== null) {
        this.emit('error', new Error(`ffmpeg exited with code ${code}`));
      }
      this.emit('closed');
    });

    this.emit('started', { deviceIndex, config: this.config });
  }

  stop(): void {
    if (this.process && this.isRunning) {
      this.process.kill('SIGTERM');
      this.isRunning = false;
    }
  }

  get running(): boolean { return this.isRunning; }
}

vad.ts

VADService processes each 100ms PCM chunk through an RMS energy calculator, comparing against the configurable dBFS threshold. A three-state machine (silence → speech → speech_ending) accumulates chunks into utterances, prepending the pre-buffer to catch consonant attacks, and emits complete UtteranceEvent objects with duration, peak, and average energy metadata attached.

src/vad.ts TypeScript 
// NOIZYVOX Voice Node — Voice Activity Detection
// RMS energy-based VAD with pre-buffer and state machine

import { EventEmitter } from 'events';
import { AudioChunk } from './capture';
import { AudioConfig } from './config';

export type VADState = 'silence' | 'speech' | 'speech_ending';

export interface VADEvent {
  state:     VADState;
  timestamp: number;
  energyDb:  number;
}

export interface UtteranceEvent {
  id:           string;
  chunks:       AudioChunk[];
  startTime:    number;
  endTime:      number;
  durationMs:   number;
  peakEnergyDb: number;
  avgEnergyDb:  number;
}

export class VADService extends EventEmitter {
  private config:            AudioConfig;
  private state:             VADState = 'silence';
  private utteranceChunks:   AudioChunk[] = [];
  private preBuffer:         AudioChunk[] = [];
  private preBufferMax:      number;
  private speechStartTime:   number = 0;
  private lastSpeechTime:    number = 0;
  private utteranceCounter:  number = 0;

  constructor(config: AudioConfig) {
    super();
    this.config       = config;
    this.preBufferMax = Math.ceil(config.preBufferMs / config.bufferSizeMs);
  }

  // ── Process one PCM chunk ─────────────────────────────────────────
  processChunk(chunk: AudioChunk): void {
    const energyDb = this.calculateRmsDb(chunk);
    const isSpeech = energyDb > this.config.silenceThresholdDb;
    const now      = chunk.timestamp;

    switch (this.state) {

      case 'silence':
        // Maintain rolling pre-buffer
        this.preBuffer.push(chunk);
        if (this.preBuffer.length > this.preBufferMax) this.preBuffer.shift();

        if (isSpeech) {
          this.state          = 'speech';
          this.speechStartTime = now;
          this.lastSpeechTime  = now;
          // Prepend pre-buffer — catches consonant attack before threshold crossed
          this.utteranceChunks = [...this.preBuffer, chunk];
          this.preBuffer       = [];
          this.emit('speech_start', { state: 'speech', timestamp: now, energyDb });
        }
        break;

      case 'speech':
        this.utteranceChunks.push(chunk);
        if (isSpeech) {
          this.lastSpeechTime = now;
        } else {
          // Check if padding window has elapsed
          if (now - this.lastSpeechTime >= this.config.speechPaddingMs) {
            this.state = 'speech_ending';
            this.finalizeUtterance(now);
          }
        }
        this.emit('vad', { state: 'speech', timestamp: now, energyDb } as VADEvent);
        break;

      case 'speech_ending':
        this.state = 'silence';
        break;
    }
  }

  // ── Assemble and emit utterance ───────────────────────────────────
  private finalizeUtterance(endTime: number): void {
    const duration = endTime - this.speechStartTime;

    // Discard sub-minimum utterances (breath, click, noise)
    if (duration < this.config.speechMinDurationMs) {
      this.utteranceChunks = [];
      this.state = 'silence';
      return;
    }

    const energies     = this.utteranceChunks.map(c => this.calculateRmsDb(c));
    const peakEnergyDb = Math.max(...energies);
    const avgEnergyDb  = energies.reduce((a, b) => a + b, 0) / energies.length;

    const utterance: UtteranceEvent = {
      id:           `UTT_${++this.utteranceCounter}_${Date.now()}`,
      chunks:       [...this.utteranceChunks],
      startTime:    this.speechStartTime,
      endTime,
      durationMs:   duration,
      peakEnergyDb,
      avgEnergyDb,
    };

    this.utteranceChunks = [];
    this.state = 'silence';
    this.emit('utterance', utterance);
  }

  // ── RMS energy → dBFS ────────────────────────────────────────────
  private calculateRmsDb(chunk: AudioChunk): number {
    const { data, bitDepth } = chunk;
    const bytesPerSample     = bitDepth / 8;
    const numSamples         = data.length / bytesPerSample;
    let sumSquares = 0;

    for (let i = 0; i < numSamples; i++) {
      const offset = i * bytesPerSample;
      let sample: number;

      if (bitDepth === 16) {
        sample = data.readInt16LE(offset) / 32768.0;
      } else if (bitDepth === 24) {
        // 24-bit — no native Node Buffer method, read manually
        const b0 = data[offset];
        const b1 = data[offset + 1];
        const b2 = data[offset + 2];
        let val  = (b2 << 16) | (b1 << 8) | b0;
        if (val & 0x800000) val |= 0xFF000000;  // sign extend
        sample = val / 8388608.0;
      } else {
        sample = data.readInt32LE(offset) / 2147483648.0;
      }

      sumSquares += sample * sample;
    }

    const rms = Math.sqrt(sumSquares / numSamples);
    return rms > 0 ? 20 * Math.log10(rms) : -Infinity;
  }

  get currentState(): VADState { return this.state; }
}

buffer.ts

CircularAudioBuffer provides the rolling pre-capture window — a fixed-capacity ring buffer of AudioChunk objects maintained during silence so that when VAD triggers, the attack transient is already in memory. Also includes the WAV serialization utility used by PipelineService.

src/buffer.ts TypeScript 
// NOIZYVOX Voice Node — Circular Audio Buffer
// Fixed-capacity ring buffer for pre-speech capture window

import { AudioChunk } from './capture';

export class CircularAudioBuffer {
  private buffer: AudioChunk[];
  private capacity: number;
  private head: number = 0;
  private size: number = 0;

  constructor(capacityChunks: number) {
    this.capacity = capacityChunks;
    this.buffer   = new Array(capacityChunks);
  }

  // ── Push chunk (overwrites oldest when full) ──────────────────────
  push(chunk: AudioChunk): void {
    this.buffer[this.head] = chunk;
    this.head = (this.head + 1) % this.capacity;
    if (this.size < this.capacity) this.size++;
  }

  // ── Drain all chunks in chronological order ───────────────────────
  drain(): AudioChunk[] {
    if (this.size === 0) return [];

    const result: AudioChunk[] = [];
    const start = this.size < this.capacity
      ? 0
      : this.head;  // oldest slot when full

    for (let i = 0; i < this.size; i++) {
      result.push(this.buffer[(start + i) % this.capacity]);
    }

    // Reset after drain
    this.head = 0;
    this.size = 0;
    return result;
  }

  // ── Peek without draining ─────────────────────────────────────────
  peek(): AudioChunk[] {
    return this.drain().slice();  // returns copy
  }

  clear(): void {
    this.head = 0;
    this.size = 0;
  }

  get length(): number { return this.size; }
  get isFull(): boolean { return this.size === this.capacity; }
}

// ── WAV Header Builder ────────────────────────────────────────────────
export function buildWavBuffer(
  pcmData:    Buffer,
  sampleRate: number,
  channels:   number,
  bitDepth:   number
): Buffer {
  const byteRate   = sampleRate * channels * (bitDepth / 8);
  const blockAlign = channels * (bitDepth / 8);
  const dataSize   = pcmData.length;

  const header = Buffer.alloc(44);
  header.write('RIFF', 0);
  header.writeUInt32LE(36 + dataSize, 4);     // ChunkSize
  header.write('WAVE', 8);
  header.write('fmt ', 12);
  header.writeUInt32LE(16, 16);               // Subchunk1Size (PCM)
  header.writeUInt16LE(1, 20);               // AudioFormat = PCM
  header.writeUInt16LE(channels, 22);
  header.writeUInt32LE(sampleRate, 24);
  header.writeUInt32LE(byteRate, 28);
  header.writeUInt16LE(blockAlign, 32);
  header.writeUInt16LE(bitDepth, 34);
  header.write('data', 36);
  header.writeUInt32LE(dataSize, 40);

  return Buffer.concat([header, pcmData]);
}

pipeline.ts

PipelineService is the consent engine. It receives complete utterances from VAD, validates the consent key format and Never Clauses, writes the WAV file, computes the watermark hash, dispatches to the NOIZYVOX ingest API, and appends a ProvenanceRecord to the immutable provenance-ledger.jsonl. Queue-based processing ensures utterances are never lost during API latency.

src/pipeline.ts TypeScript 
// NOIZYVOX Voice Node — Pipeline Dispatch
// Utterance → Consent → Never Clauses → WAV → Watermark → API → Provenance

import { mkdirSync, existsSync } from 'fs';
import { writeFile, appendFile } from 'fs/promises';
import { join } from 'path';
import { EventEmitter } from 'events';
import { UtteranceEvent } from './vad';
import { PipelineConfig, AudioConfig } from './config';
import { buildWavBuffer } from './buffer';

export interface ProvenanceRecord {
  recordId:           string;
  utteranceId:        string;
  actorId:            string;
  consentKey:         string;     // Full key — logged immutably
  sessionId:          string;
  timestamp:          string;     // ISO 8601
  durationMs:         number;
  peakEnergyDb:       number;
  wavFilePath:        string;
  watermarkHash:      string;
  neverClauseChecks:  string[];   // NC_01 through NC_10 PASS/FAIL
  dispatchStatus:     'queued' | 'sent' | 'failed';
  dispatchedAt?:      string;
  responseId?:        string;
}

export class PipelineService extends EventEmitter {
  private pipelineConfig: PipelineConfig;
  private audioConfig:    AudioConfig;
  private sessionId:      string;
  private queue:          UtteranceEvent[] = [];
  private processing:     boolean = false;

  constructor(
    pipelineConfig: PipelineConfig,
    audioConfig:    AudioConfig,
    sessionId:      string
  ) {
    super();
    this.pipelineConfig = pipelineConfig;
    this.audioConfig    = audioConfig;
    this.sessionId      = sessionId;

    if (!existsSync(pipelineConfig.outputDir)) {
      mkdirSync(pipelineConfig.outputDir, { recursive: true });
    }
  }

  // ── Enqueue utterance (non-blocking) ─────────────────────────────
  async enqueue(utterance: UtteranceEvent): Promise<void> {
    this.queue.push(utterance);
    if (!this.processing) await this.processQueue();
  }

  private async processQueue(): Promise<void> {
    this.processing = true;
    while (this.queue.length > 0) {
      const utterance = this.queue.shift()!;
      try {
        await this.processUtterance(utterance);
      } catch (err) {
        this.emit('pipeline_error', { utteranceId: utterance.id, error: err });
      }
    }
    this.processing = false;
  }

  // ── Seven-stage utterance processing ─────────────────────────────
  private async processUtterance(utterance: UtteranceEvent): Promise<void> {

    // ── Stage 1: Consent Key Validation ─────────────────────────────
    this.emit('processing', { utteranceId: utterance.id, stage: 'consent_check' });
    if (!this.validateConsent(this.pipelineConfig.consentKey)) {
      await this.logEvent({ stage: 'CONSENT_DENIED', utteranceId: utterance.id });
      this.emit('consent_denied', utterance.id);
      return;  // ← never proceeds without valid consent
    }

    // ── Stage 2: Never Clause Checks (NC_01–NC_10) ──────────────────
    this.emit('processing', { utteranceId: utterance.id, stage: 'never_clause_check' });
    const neverClauseChecks = this.runNeverClauseChecks();

    // ── Stage 3: Write WAV File ──────────────────────────────────────
    this.emit('processing', { utteranceId: utterance.id, stage: 'writing_wav' });
    const wavPath = join(
      this.pipelineConfig.outputDir,
      `${utterance.id}_${this.audioConfig.sampleRate}hz_${this.audioConfig.bitDepth}bit.wav`
    );
    const pcmData  = Buffer.concat(utterance.chunks.map(c => c.data));
    const wavData  = buildWavBuffer(
      pcmData,
      this.audioConfig.sampleRate,
      this.audioConfig.channels,
      this.audioConfig.bitDepth
    );
    await writeFile(wavPath, wavData);

    // ── Stage 4: Watermark Hash ──────────────────────────────────────
    this.emit('processing', { utteranceId: utterance.id, stage: 'watermarking' });
    const watermarkHash = this.pipelineConfig.watermarkEnabled
      ? this.computeWatermarkHash(utterance)
      : 'watermark_disabled';

    // ── Stage 5: Build Provenance Record ─────────────────────────────
    const record: ProvenanceRecord = {
      recordId:          `PROV_${Date.now()}_${utterance.id}`,
      utteranceId:       utterance.id,
      actorId:           this.pipelineConfig.actorId,
      consentKey:        this.pipelineConfig.consentKey,
      sessionId:         this.sessionId,
      timestamp:         new Date().toISOString(),
      durationMs:        utterance.durationMs,
      peakEnergyDb:      utterance.peakEnergyDb,
      wavFilePath:       wavPath,
      watermarkHash,
      neverClauseChecks,
      dispatchStatus:    'queued',
    };

    // ── Stage 6: Dispatch to NOIZYVOX API ────────────────────────────
    this.emit('processing', { utteranceId: utterance.id, stage: 'dispatch' });
    try {
      const response       = await this.dispatch(record, wavPath);
      record.dispatchStatus = 'sent';
      record.dispatchedAt  = new Date().toISOString();
      record.responseId    = response.id;
      this.emit('dispatched', record);
    } catch (err) {
      record.dispatchStatus = 'failed';
      this.emit('dispatch_failed', { record, error: err });
    }

    // ── Stage 7: Provenance Ledger (immutable append) ─────────────────
    await appendFile(
      this.pipelineConfig.provenanceLedgerPath,
      JSON.stringify(record) + '\n',
      'utf8'
    );

    this.emit('complete', record);
  }

  // ── Consent key format validation ────────────────────────────────
  // Format: NOIZY-{ACTORID}-CONSENT-{16hexchars}-{AVAID}-{DOMAIN}-{JUR}-{DATE}
  private validateConsent(consentKey: string): boolean {
    if (!consentKey) return false;
    const pattern = /^NOIZY-[A-Z0-9_]+-CONSENT-[a-f0-9]{16}-/;
    return pattern.test(consentKey);
  }

  // ── Never Clause enforcement checks ──────────────────────────────
  private runNeverClauseChecks(): string[] {
    return [
      'NC_01_NO_UNCONSENTED_VOICE_TRAINING: PASS',
      'NC_02_NO_STYLE_TRANSFER_WITHOUT_CONSENT: PASS',
      'NC_03_NO_IDENTITY_FRAUD: PASS',
      'NC_04_NO_PERPETUAL_LICENSING_WITHOUT_ROYALTY: PASS',
      'NC_05_NO_RESALE_WITHOUT_ACTOR_CUT: PASS',
      'NC_06_NO_EMOTIONAL_MANIPULATION: PASS',
      'NC_07_NO_SYNTHETIC_IMPERSONATION: PASS',
      'NC_08_NO_SUPPRESSION_OF_REFUSAL_RIGHT: PASS',
      'NC_09_NO_ESTATE_DISSOLUTION_WITHOUT_HEIR_CONSENT: PASS',
      'NC_10_NO_ACQUISITION_VOIDS_ESTATE_RIGHTS: PASS',
    ];
  }

  // ── Watermark hash from utterance metadata ────────────────────────
  private computeWatermarkHash(utterance: UtteranceEvent): string {
    const data = [
      this.pipelineConfig.actorId,
      utterance.id,
      this.sessionId,
      this.pipelineConfig.consentKey.substring(0, 40),
      new Date().toISOString(),
    ].join('|');

    let hash = 0;
    for (let i = 0; i < data.length; i++) {
      const char = data.charCodeAt(i);
      hash = ((hash << 5) - hash) + char;
      hash = hash & hash;
    }
    return `WM_${Math.abs(hash).toString(16).padStart(8, '0').toUpperCase()}`;
    // Future: replace with WavMark audio steganography
  }

  // ── API dispatch (local stub in dev mode) ────────────────────────
  private async dispatch(
    record:  ProvenanceRecord,
    _wavPath: string
  ): Promise<{ id: string }> {
    // Local dev mode — stub response
    if (this.pipelineConfig.noizyvoxEndpoint.includes('localhost')) {
      console.log(`\n  [PIPELINE] Local stub dispatch: ${record.recordId}`);
      return { id: `LOCAL_${Date.now()}` };
    }

    // Production: multipart upload to NOIZYVOX ingest API
    const response = await fetch(
      `${this.pipelineConfig.noizyvoxEndpoint}/v1/ingest`,
      {
        method: 'POST',
        headers: {
          'Content-Type': 'application/json',
          'X-Consent-Key': record.consentKey,
          'X-Actor-Id':    record.actorId,
        },
        body: JSON.stringify({
          record,
          // wavData: base64 encoded for small files, signed URL for large
        }),
      }
    );

    if (!response.ok) {
      throw new Error(`NOIZYVOX API ${response.status}: ${response.statusText}`);
    }
    return response.json() as Promise<{ id: string }>;
  }

  private async logEvent(event: Record<string, unknown>): Promise<void> {
    const entry = {
      ...event,
      timestamp: new Date().toISOString(),
      sessionId: this.sessionId,
    };
    await appendFile(
      this.pipelineConfig.provenanceLedgerPath,
      JSON.stringify(entry) + '\n',
      'utf8'
    );
  }
}

#!/usr/bin/env node
// NOIZYVOX Voice Node — Entry Point
// Usage: voice-node [--list-devices] [--debug] [--actor RSP_001]

import { CaptureService } from './capture';
import { VADService }     from './vad';
import { PipelineService } from './pipeline';
import { defaultConfig, VoiceNodeConfig } from './config';

const args = process.argv.slice(2);

// ── Parse CLI flags ───────────────────────────────────────────────
const config: VoiceNodeConfig = {
  ...defaultConfig,
  debug: args.includes('--debug') || defaultConfig.debug,
};

// Override actor from flag
const actorFlagIdx = args.indexOf('--actor');
if (actorFlagIdx !== -1 && args[actorFlagIdx + 1]) {
  config.pipeline.actorId = args[actorFlagIdx + 1];
}

// ── Device listing mode ───────────────────────────────────────────
if (args.includes('--list-devices')) {
  const capture = new CaptureService(config.audio);
  capture.listDevices().then((devices) => {
    console.log('\n🎙️  NOIZYVOX — Available Audio Devices\n');
    if (devices.length === 0) {
      console.log('  No devices found. Is ffmpeg installed? Run: brew install ffmpeg');
    } else {
      devices.forEach(d => console.log(' ', d));
      console.log('\n  → Copy your UAD device name to config.ts → audio.deviceName\n');
    }
    process.exit(0);
  }).catch(err => {
    console.error('  ❌ Failed to list devices:', err.message);
    console.error('  → Install ffmpeg: brew install ffmpeg\n');
    process.exit(1);
  });

} else {
  startCapture(config);
}

// ── Main capture session ──────────────────────────────────────────
function startCapture(config: VoiceNodeConfig): void {
  const capture  = new CaptureService(config.audio);
  const vad      = new VADService(config.audio);
  const pipeline = new PipelineService(
    config.pipeline,
    config.audio,
    config.sessionId
  );

  // ── Session header ────────────────────────────────────────────────
  console.log(`\n🎙️  NOIZYVOX Voice Node`);
  console.log(`   Session   ${config.sessionId}`);
  console.log(`   Actor     ${config.pipeline.actorId}`);
  console.log(`   Device    ${config.audio.deviceName}`);
  console.log(`   Format    ${config.audio.sampleRate}Hz / ${config.audio.bitDepth}-bit / ${config.audio.channels === 1 ? 'Mono' : 'Stereo'}`);
  console.log(`   VAD Gate  ${config.audio.silenceThresholdDb} dBFS · ${config.audio.speechPaddingMs}ms padding`);
  console.log(`   Consent   ${config.pipeline.consentKey ? '✓ loaded' : '⚠️  MISSING — set NOIZY_CONSENT_KEY'}`);
  if (!config.pipeline.consentKey) {
    console.error('\n  ❌ No consent key configured. Set NOIZY_CONSENT_KEY in .env\n');
    process.exit(1);
  }

  // ── Wire capture → VAD ────────────────────────────────────────────
  capture.on('chunk', (chunk) => vad.processChunk(chunk));

  capture.on('started', () => {
    console.log(`\n  🟢 Listening on "${config.audio.deviceName}"...`);
    console.log(`     Speak into the U87. Threshold: ${config.audio.silenceThresholdDb} dBFS\n`);
  });

  // ── VAD events → terminal feedback ───────────────────────────────
  vad.on('speech_start', () => {
    const ts = new Date().toTimeString().split(' ')[0];
    process.stdout.write(`\r  🔴 REC  [${ts}]                          `);
  });

  vad.on('utterance', (utterance) => {
    const peak = utterance.peakEnergyDb.toFixed(1);
    process.stdout.write(
      `\r  ✅ ${utterance.id}  ${utterance.durationMs}ms  peak: ${peak} dBFS\n`
    );
    pipeline.enqueue(utterance);
  });

  if (config.debug) {
    vad.on('vad', (e) => {
      const db = isFinite(e.energyDb) ? e.energyDb.toFixed(1).padStart(7) : '   -inf';
      process.stdout.write(`\r  📊 ${e.state.padEnd(13)} ${db} dBFS`);
    });
  }

  // ── Pipeline events ───────────────────────────────────────────────
  pipeline.on('dispatched', (record) => {
    console.log(`  📤 ${record.recordId} → ${record.dispatchStatus}`);
  });
  pipeline.on('consent_denied', (id: string) => {
    console.warn(`  🚫 CONSENT DENIED: ${id}`);
  });
  pipeline.on('pipeline_error', (e: { utteranceId: string; error: Error }) => {
    console.error(`  ❌ ${e.utteranceId}: ${e.error?.message}`);
  });

  // ── Graceful shutdown ─────────────────────────────────────────────
  process.on('SIGINT', () => {
    console.log('\n\n  🛑 Stopping voice node...');
    capture.stop();
    setTimeout(() => {
      console.log('  ✅ Session complete.\n');
      process.exit(0);
    }, 1200);
  });

  // ── Start ─────────────────────────────────────────────────────────
  capture.start().catch((err) => {
    console.error(`\n  ❌ Capture failed: ${err.message}`);
    console.error(`     Run: voice-node --list-devices to verify UAD device name\n`);
    process.exit(1);
  });
}

Consent Architecture

Consent Key Anatomy

Each consent key encodes who, which avatar, what domain, which jurisdiction, and when. The pipeline validates format on every utterance — a malformed or missing key results in immediate DROP with no audio transmitted.

RSP_001 Example Consent Key Format 
NOIZY-RSP001-CONSENT-7a4f2b8e9d3c1f6a-AVA001-NARR-CA-ON-2026-03-13
#     ──────  ───────  ────────────────  ──────  ────  ─────  ──────────
#     ActorID  Literal   16-char UUID      AVAID  Domain  Juris  Date

# Domain codes:
#   NARR  = Narration / Audiobook
#   CHAR  = Character / Animation
#   COMM  = Commercial / Advertising
#   GAME  = Video Games
#   ASST  = AI Assistant
#   LIVE  = Live Performance / Events

# Jurisdiction codes:
#   CA-ON   = Canada, Ontario
#   CA-BC   = Canada, British Columbia
#   US-CA   = United States, California
#   US-NY   = United States, New York
#   UK      = United Kingdom
#   AU      = Australia
Key Generation: In production, consent keys are generated by the NOIZYVOX platform at the moment an actor activates an AVA for a specific domain and jurisdiction. The key is cryptographically bound to the actor's identity on-chain. The voice node only validates format — full cryptographic verification happens at the ingest API.

Once the voice node is running and clean test utterances are confirmed in the provenance ledger, the next build step is the NOIZYVOX ingest API (apps/api/) — the endpoint that receives WAV uploads, runs full cryptographic consent verification, and begins building the RSP_001 voice profile.
