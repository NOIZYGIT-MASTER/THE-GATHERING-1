# Nerve-to-Note

## Summary

Nerve-to-Note is the performance interface that translates human physical intent into expressive musical control without flattening the artist's human signature.

## Current Build Order

1. Raw sensor ingestion
2. Feature extraction
3. Intent vector
4. Deterministic mapping
5. Neural smoothing
6. Non-linear quantization
7. MIDI or MPE output

## Non-Negotiables

- Map intent before smoothing
- Preserve expressive instability where it is musically meaningful
- Keep raw biometric data local
- Separate tremor filtering from erasure of identity
- Allow mapping targets beyond Western defaults, including world scale systems and rhythm logics

## Open Questions

- What fields belong in the intent vector?
- What is the boundary between assistive correction and authorship change?
- How should corrected gestures be reflected in provenance?
